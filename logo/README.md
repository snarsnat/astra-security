# ASTRA Logo

## Logo Concept

The ASTRA logo represents:
- **A** - Advanced Security
- **Stars** - Guidance, Protection, Navigation
- **Shield** - Security, Defense
- **Orbital Rings** - Continuous Protection, 360° Security
- **Blue Color** - Trust, Technology, Reliability

## Logo Files

- `astra-logo.svg` - Vector logo (primary)
- `astra-logo.png` - High-resolution PNG (256x256, 512x512, 1024x1024)
- `astra-icon.png` - Favicon (32x32, 64x64)
- `astra-logo-dark.svg` - Dark mode variant

## Usage Guidelines

### Primary Logo
Use the primary logo (blue) for most applications.

### Dark Mode
Use the dark mode variant on dark backgrounds.

### Minimum Size
- Full logo: 100px width minimum
- Icon only: 32px width minimum

### Clear Space
Maintain clear space equal to the height of the "A" character around the logo.

### Color Variations
- **Primary**: #2563eb (blue)
- **Dark**: #1e40af (dark blue)
- **Light**: #60a5fa (light blue)
- **Monochrome**: #ffffff (white) or #000000 (black)

## Typography

- **Primary Font**: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **Monospace Font**: "SF Mono", Monaco, "Courier New", monospace (for code/technical)

## Brand Voice

- **Trustworthy but friendly**
- **Advanced but accessible**
- **Secure but unobtrusive**
- **Professional but human**

## Do Not

- ❌ Stretch or distort the logo
- ❌ Change the colors arbitrarily
- ❌ Add effects (drop shadows, glows)
- ❌ Place on busy backgrounds
- ❌ Use outdated versions
